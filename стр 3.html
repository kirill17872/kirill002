NVIDIA Quadro GV100 GPU

Легкая реализация с использованием множества API-интерфейсов. Разработчики могут получить доступ к технологии NVIDIA RTX через интерфейс программирования приложений NVIDIA OptiX, новый API DirectX Raytracing от Microsoft и, в будущем, Vulkan, открытый межплатформенный графический стандарт. Все три API имеют общую модель программирования шейдеров, которая позволяет разработчикам поддерживать несколько платформ.
